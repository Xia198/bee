# 🐝 九种蜜蜂属图鉴

一个关于蜜蜂的课程拓展作业，展示九种蜜蜂属的精美图鉴。

## 特点

- 🎨 温暖手绘风格设计
- ✨ 丰富的动画效果
- 🖼️ 精美的卡片布局
- 📱 响应式设计，支持各种设备
- 💫 点击交互展示详情

## 九种蜜蜂属

1. 西方蜜蜂 (Apis mellifera)
2. 东方蜜蜂 (Apis cerana)
3. 大蜜蜂 (Apis dorsata)
4. 小蜜蜂 (Apis florea)
5. 黑小蜜蜂 (Apis andreniformis)
6. 吕宋蜂 (Apis luzonica)
7. 巴塔哥尼亚蜜蜂 (Apis roratopsanky)
8. 印度大蜜蜂 (Apis laboriosa)
9. 沙巴蜂 (Apis koschevnikovi)

## 技术栈

- HTML5
- CSS3
- JavaScript
- Google Fonts (马善政字体)

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。

或使用 Python 启动本地服务器：

```bash
python -m http.server 8000
```

然后访问 http://localhost:8000
