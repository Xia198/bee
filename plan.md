# 🐝 蜜蜂图鉴网站开发计划

## 项目概述

**项目名称**：Nine Bees - 九种蜜蜂属图鉴  
**项目类型**：课程拓展作业 / 个人展示网站  
**目标用户**：课程评审、同学、老师  
**核心目标**：展示9种蜜蜂属的精美图鉴，以视觉美学为核心，简单点击交互展示详情  

---

## 一、规划与设计阶段

### 1.1 需求确认
- [x] 确定核心功能：9种蜜蜂属展示 + 点击查看详情
- [x] 确定重点：视觉美化
- [x] 简化交互：无需复杂功能，聚焦展示效果

### 1.2 内容规划
- [ ] 确定9种蜜蜂属名单（附学名）
  1. 西方蜜蜂 (Apis mellifera)
  2. 东方蜜蜂 (Apis cerana)
  3. 大蜜蜂 (Apis dorsata)
  4. 小蜜蜂 (Apis florea)
  5. 黑小蜜蜂 (Apis andreniformis)
  6. 吕宋蜂 (Apis luzonica)
  7. 巴塔哥尼亚蜜蜂 (Apis roratopsanky)
  8. 印度大蜜蜂 (Apis laboriosa)
  9. 沙巴蜂 (Apis koschevnikovi)
- [ ] 收集每种蜜蜂的资料（分布区域、特征、习性）

### 1.3 视觉设计
- [ ] 确定整体风格方向
  - 风格选项：
    - A. 清新自然风（绿叶、阳光、花朵元素）
    - B. 学术图鉴风（复古纸张、科学插画风格）
    - C. 现代简约风（大片留白、精致排版）
    - D. 温暖手绘风（手绘插画、水彩质感）
- [ ] 设计配色方案
  - 主色调：蜂蜜金 / 琥珀色
  - 背景：奶油白 / 淡米色
  - 点缀：蜂巢棕 / 深棕色
- [ ] 设计版式布局
  - 瀑布流/网格展示
  - 卡片式设计
  - 详情弹窗/侧边栏

## 技术方案

| 类别 | 选择 | 说明 |
|------|------|------|
| 核心 | HTML5 + CSS3 + JavaScript | 轻量、通用 |
| CSS | 原生 CSS / Tailwind CSS | 灵活控制视觉效果 |
| 动画 | CSS Animation + Intersection Observer | 流畅的视觉体验 |
| 图片 | Unsplash / Pexels 免费图库 | 高质量免费素材 |
| 部署 | GitHub Pages / Vercel | 免费快速 |

- 标题：Playfair Display / Noto Serif SC（衬线体，优雅学术感）
- 正文：Inter / Noto Sans SC（无衬线，易读）
